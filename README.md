# boxicons-quasar

This package exports all boxicons path strings for usage inside quasar q-icon component.

### Usage

```bash
yarn add boxicons-quasar
```

```vue
<script setup lang="ts">
import { bxExit } from 'boxicons-quasar';
</script>

<template>
  <q-icon :name="bxExit" size="24px" />
</template>
```
